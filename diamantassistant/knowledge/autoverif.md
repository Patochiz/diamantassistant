# Regles de verification avant validation

## Orthographe
- Verifier les fautes d'orthographe dans les descriptions des lignes
- Signaler les mots mal orthographies

## Coherence des produits
- Si un produit principal est present, verifier que ses accessoires habituels sont aussi dans le document
- Signaler si une ligne semble manquer

## Quantites et prix
- Verifier que les quantites ne semblent pas aberrantes
- Signaler les prix unitaires a 0

## Informations manquantes
- Verifier que la reference client est renseignee
- Verifier que la date de livraison est presente
